# Checklist Search Console et Bing Webmaster Tools

À exécuter après une release stable :

1. vérifier la propriété domaine `fps-airsoft-trajectory.com` dans Google Search
   Console et Bing Webmaster Tools ;
2. soumettre `https://fps-airsoft-trajectory.com/sitemap.xml` puis contrôler sa
   dernière lecture et les URL exclues ;
3. inspecter la home, le convertisseur Joules/FPS, l’outil gaz, le simulateur 3D,
   la FAQ et les quatre guides ;
4. confirmer la canonique HTTPS sans `www`, le rendu HTML et les données
   structurées détectées ;
5. surveiller indexation, erreurs d’exploration, Core Web Vitals et améliorations
   structurées ;
6. comparer requêtes, impressions, position moyenne et CTR avant toute décision
   éditoriale ;
7. garder la galerie en `noindex,follow` hors sitemap jusqu’à ce que ses premières
   cards publiées soient présentes dans un HTML crawlable avec pagination et
   `ItemList` ;
8. ne demander l’indexation d’une future card que si elle possède une URL stable,
   une modération validée, un texte utile, une canonique et une image sociale.

Le code ne permet pas, à lui seul, de conclure à une amélioration de position.
