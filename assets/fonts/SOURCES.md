# Polices auto-hébergées

Les fichiers servis par F.A.T. sont locaux ; aucune police n’est chargée depuis un CDN en production.

- **Inter** : fichier déjà présent dans le projet ; licence officielle conservée depuis <https://github.com/rsms/inter/blob/master/LICENSE.txt>.
- **Saira** : sous-ensemble latin WOFF2 servi par Google Fonts (`fonts.gstatic.com`), avec source et licence dans <https://github.com/google/fonts/tree/main/ofl/saira>.
- **Saira Stencil One** : sous-ensemble latin WOFF2 servi par Google Fonts (`fonts.gstatic.com`), avec source et licence dans <https://github.com/google/fonts/tree/main/ofl/sairastencilone>.
- **IBM Plex Mono** : sous-ensembles Latin1 officiels Regular, Medium et SemiBold de <https://github.com/IBM/plex/tree/master/packages/plex-mono/fonts/split/woff2>.

Les copies des licences SIL Open Font License 1.1 sont placées dans `licenses/`.
